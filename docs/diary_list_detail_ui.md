# 📓 일기 앱 UI 가이드 --- 목록 & 상세 페이지

본 문서는 **일기 목록 페이지**와 **일기 상세 페이지**의 UI 구조와 색상
조합만을 정의한다.\
캐릭터 요소는 포함되지만, 이모지 사용은 최소화한다.

------------------------------------------------------------------------

# 🎯 공통 디자인 방향

## 키워드

-   Soft
-   Healing
-   Premium
-   Pastel

## 기본 스타일 규칙

-   카드형 레이아웃
-   radius: 20\~28
-   그림자 약하게 (blur 16\~24 / opacity 4\~6%)
-   배경: #F8F9FC 계열
-   기본 텍스트: #1F2937
-   서브 텍스트: #6B7280

# 📋 일기 목록 페이지 UI

------------------------------------------------------------------------

## 🔲 카드 기본 구조

    [ 감정 캐릭터 ]

    제목
    요약 한 줄

    날짜 · 시간

    [ 작성방식 태그 ]

------------------------------------------------------------------------

## 📐 레이아웃

-   좌측: 감정 캐릭터 (48\~56px)
-   중앙: 제목 / 요약
-   우측 상단: 작성방식 태그
-   전체 카드 padding: 16\~20

------------------------------------------------------------------------

## 🎨 카드 배경 컬러

-   기본 카드: White (#FFFFFF)
-   감정 포인트 컬러는 아주 연하게 상단 그라데이션으로 사용

  감정   포인트 컬러
  ------ -------------
  설렘   #FFD6C9
  기쁨   #FFE8A3
  평온   #CFF5E7
  슬픔   #DCE9FF
  분노   #E7D9FF

------------------------------------------------------------------------

## 🏷️ 태그 스타일

### 작성 방식

-   AI: 연보라 배경 / #8B7CF6 텍스트
-   자유: 민트 배경 / #4CC9A6 텍스트

radius: 999\
padding: horizontal 10 / vertical 4

------------------------------------------------------------------------

## ➕ 플로팅 버튼

-   위치: 우하단
-   크기: 56
-   색상: Primary Purple (#8B7CF6)
-   그림자 강조

------------------------------------------------------------------------

# 📖 일기 상세 페이지 UI

------------------------------------------------------------------------

## 🔝 상단 히어로 영역

구성: - 감정 캐릭터 크게 표시 - 감정명 - 날짜 · 시간 - 작성 방식 태그

스타일: - 카드형 - 감정 컬러 기반 그라데이션 배경 - radius 28

------------------------------------------------------------------------

## 📝 제목 영역

-   위치: 히어로 아래
-   font-size: 22\~24
-   font-weight: 600+
-   색상: #111827

------------------------------------------------------------------------

## ✍️ 본문 영역

-   카드 컨테이너
-   background: #FFFFFF
-   line-height: 1.6
-   radius: 24

------------------------------------------------------------------------

## 🖼️ 이미지 / 그림 섹션

-   가로 스크롤 카드
-   한 장 크기: 120\~160
-   radius: 20
-   종이 느낌 그림자

------------------------------------------------------------------------

## 🤖 AI 조언 카드

-   기본 접힘 상태
-   배경: #F1EEFF

-   아이콘 작게 사용
